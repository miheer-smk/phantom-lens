# Phantom Lens

**Project:** Physics-Anchored Deepfake Detection (PRISM)
**Researcher:** Miheer Satish Kulkarni  
**Institution:** IIIT Nagpur  
**Year:** 2026  

## About
Phantom Lens is a deepfake detection framework built on 24 physics-based 
feature pillars including shadow geometry, sensor noise, and boundary 
analysis. Developed independently as original research work.
